V2.1.1 / 27-July-2018

Main Changes

  • Release notes update to new format


Update History


V2.1.0 / 17-February-2017

Main Changes

  • stm32l476g_eval_sd.c/.h
    • Add weak BSP SD functions
      • BSP_SD_MspInit(), BSP_SD_MspDeInit(), BSP_SD_WriteCpltCallback(), BSP_SD_ReadCpltCallback() and BSP_SD_AbortCallback()

V2.0.0 / 17-February-2017

Main Changes

  • stm32l476g_eval_sd.c/.h
    • Alignment with new SD HAL APIs interface
  • stm32l476g_eval_qspi.c
    • Update BSP_QSPI_Write() to fasten write address calculation
  • NOTE: This release must be used with 
    • STM32L4xx HAL/LL driver V1.7.0 or newer

V1.0.4 / 08-April-2016

Main Changes

  • stm32l476g_eval_audio.c/.h
    • Apply DFSDM renaming
      • DFSDM_Channel3 and DFSDM_Channel4 renamed respectively to DFSDM1_Channel3 and DFSDM1_Channel4
      • DFSDM_Filter0 and DFSDM_Filter1 renamed respectively to DFSDM1_Filter0 and DFSDM1_Filter1
      • __HAL_RCC_DFSDM_CLK_ENABLE() and __HAL_RCC_DFSDM_CLK_DISABLE() renamed respectively to __HAL_RCC_DFSDM1_CLK_ENABLE() and __HAL_RCC_DFSDM1_CLK_DISABLE()
      • GPIO_AF6_DFSDM renamed to GPIO_AF6_DFSDM1
  • stm32l476g_eval_sd.c
    • Update SD_MspInit() to retrieve RCC configuration for PLLSAI1 source clock and divider M
  • NOTE: This release must be used with STM32L4xx HAL driver V1.5.0 or later

V1.0.3 / 29-January-2016

Main Changes

  • stm32l476g_eval.c
    • Update FMC timings for LCD access in FMC_BANK3_Init()
  • stm32l476g_eval_qspi.c
    • Deactivate automatic timeout in memory mapped mode
  • NOTE: This release must be used with STM32L4xx HAL driver V1.2.0 or later

V1.0.2 / 25-November-2015

Main Changes

  • stm32l476g_eval.c
    • Remove FMC write Fifo feature configuration in FMC_BANK3_Init() as not available
  • stm32l476g_eval_audio.c
    • Apply correction in BSP_AUDIO_OUT_Init() on SAI clock strobing in line with HAL SAI correction
  • stm32l476g_eval_idd.c
    • Change call to HAL_PWREx_EnterSTOP1Mode() after stop 0 addition and HAL_PWREx_EnterSTOP1Mode() API update
  • stm32l476g_eval_nor.c
    • Fix FMC timings on Address setup time and Data setup time in BSP_NOR_Init()
  • stm32l476g_eval_sram.c
    • Fix FMC timings on Address setup time, Data setup time and Bus Turn Around duration in BSP_SRAM_Init()
  • NOTE: This release must be used with STM32L4xx HAL driver V1.2.0 or later

V1.0.1 / 16-September-2015

Main Changes

  • stm32l476g_eval_qspi.c
    • Set QuadSPI prescaler to 1 to configure interface at 40Mhz (in line with component datasheet)
  • stm32l476g_eval.c, stm32l476g_eval_audio.c, stm32l476g_eval_idd.c, stm32l476g_eval_lcd.c,  stm32l476g_eval_nor.c, stm32l476g_eval_qspi.c, stm32l476g_eval_sd.c and stm32l476g_eval_sram.c
    • Apply new GPIO speed frequency define from HAL GPIO (GPIO_SPEED_FREQ_xxxx)
  • NOTE: This release must be used with STM32L4xx HAL driver V1.1.0 or later

V1.0.0 / 26-June-2015

Main Changes

  • First official release of STM32L476G_EVAL board drivers for STM32Cube L4 FW package
